# qus 个人介绍页

基于 Next.js + Aceternity UI 的个人介绍网站。

## 技术栈

- Next.js 15
- React 19
- Tailwind CSS
- Motion (Framer Motion)
- Aceternity UI 组件

## 设计规范 (AGENTS.md)

- 反主流美学，避免紫/靛蓝配色
- 不对称布局，避免 Hero+三卡片
- 噪点纹理或渐变背景
- 简短、主动语态文案

## 本地运行

```bash
npm install
npm run dev
```

访问 http://localhost:3000

## 自定义

- 修改 `src/app/page.tsx` 中的邮箱、B站链接等
- 修改 `AGENTS.md` 了解设计约束
